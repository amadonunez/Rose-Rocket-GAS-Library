# Rose Rocket GAS Library\n\nEste es el repositorio de una biblioteca de Google Apps Script.\n\n**Funcionalidad Principal:**\nEsta biblioteca contiene una colección de funciones reutilizables diseñadas para interactuar con la API de Rose Rocket. Su propósito es ser utilizada por otros proyectos de Apps Script, proporcionando una forma centralizada y eficiente de manejar las solicitudes y la autenticación, lo que evita la duplicación de código.
